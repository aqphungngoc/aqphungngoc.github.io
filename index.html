Dev App Engine Development & Testing Only

Dev App Engine is a personal development and testing application maintained by aqphungngoc. It is used exclusively for experimenting with APIs, OAuth integrations, and other development workflows. This application is not intended for commercial use and is not available to the general public.

Purpose

This app serves as a sandbox environment for:

  • Testing OAuth 2.0 authentication flows (e.g., Google OAuth)
  • Experimenting with third-party API integrations
  • Prototyping and evaluating new development ideas

No user data is collected, stored, or shared for any commercial or production purpose. Any data accessed during testing is used solely to verify that integrations work correctly and is handled in accordance with our Privacy Policy.
